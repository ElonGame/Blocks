This is a project that I work on very sporadically when I have the time and energy to do so. In the past, development has ceased for entire months at a time. This is just a hobby of mine, and so time gets spent where I think is interesting. Much of the code may be very messy or inefficient. Regardless, the code does work, and works moderately well. Blocks has been a great learning experience for me, and I hope that it can help others as well.

Compiling and running

NixOS

I run NixOS. Because of this, I have set up a script to launch a nix-shell with all of the requirements.

  • git clone https://github.com/Pleune/Blocks
  • cd Blocks
  • ./run
Linux
  • Make sure SLD2 SDL2_tf and glew are installed.
  • git clone https://github.com/Pleune/Blocks
  • cd Blocks
  • Compile with make
  • Run with ./bin/blocks
